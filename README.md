# Node-TypeScript-TypeORM-Boilerplate

RESTful API boilerplate with JWT authentication using the following technologies:

- Express
- TypeScript
- TypeORM
- Postgres
- Docker

## Requirements

- Node
- Docker

## Running on Local

- clone the repo
- **npm install**
- **npm start**

## Running in a Dockerized Container

- Clone the repo
- **npm run docker:dev**
